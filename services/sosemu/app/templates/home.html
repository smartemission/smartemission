HTML Rendering

Deze API kan ook met HTML responses ipv JSON bevraagd worden, via de optionele format=html query parameter. Hieronder de voorbeelden van hierboven als HTML.

  • Stations: {{ url_for('stations') }}?format=html
  • Laatste metingen per station (bijv station 23): {{ url_for('timeseries') }}?station=23&format=html

Web Applicatie

Deze web applicatie is geschreven in Python met het lichtgewicht Flask web-framework. Zie ook de webapp code in GitHub.

JSONP Support

JSONP support is via the callback parameter, for example: http://data.smartemission.nl/sosemu/api/v1/stations?callback=mycallback
